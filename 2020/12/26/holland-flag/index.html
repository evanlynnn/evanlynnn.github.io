荷兰国旗问题

题目描述

现有n各红白蓝三种不同颜色的小球，乱序排列在一起，请通过两两交换任意两个小球，使得从左至右，依次是一些红球、一些白球、一些蓝球。
红白蓝分别编号为0、1、2
LeetCode:https://leetcode-cn.com/problems/sort-colors/

分析与解法

这个问题类似于排序，目标是将乱序的数组排序，划分为0、1、2三部分。
因为一共只有三个部分，只要前面的部分（1）和后面的部分（2）划分好，中间部分（1）就自然出来了。
利用三指针，我们可以经过一趟扫描就实现。时间复杂度O(n)

三指针


我们设三个指针：begin、current、end，其中：
begin指针以前的数都是0。
current指针指向当前待排序的数，begin和current之间的数为1。
end指针之后的数都是2.
初始状态下：begin和current都指向第一个元素（下标0），end指向最后一个元素（下标n-1）。
下面我开始对数组进行扫描：
每一次我们判断current指向的数。

  1. current指向的数为0
    0是属于前面的部分，他应该在begin指针之前。所以我们要将0移入第一部分，并将begin、current指针后移：
    swap(colors[begin], colors[end]);
begin++;
current++;
  2. current指向的数为1
    因为1是在begin和current之间，所以只需要将current后移就行，不需要交换操作。
  3. current指向的数为2
    2属于最后的部分，它应该位于end指针之后。
    现在我们交换current和end指针指向的数，再将end指针前移，就完成了插入。
    那current需要后移吗？

不需要！原先end指针指向的数可能是1，也可能是0。如果为0的话，current后移会使中间部分无序。

swap(colors[current], colors[end]);
end--;

代码

//完整C++
#include <iostream>
#include <algorithm>
using namespace std;
int n;
const int MAXN = 1000;
int colors[MAXN];
/**
 * Partition the array into three sections: all-0, all-1, all-2 
 *
 */
void partition(int n) {
    //Let begin point to the first element, and end point to the last element.
    int begin = 0;
    int current = 0;
    int end = n - 1;
    while(current <= end) {
        if(colors[current] == 0) {
            swap(colors[current], colors[begin]);
            begin++;
            current++;
        } else if(colors[current] == 2) {
            swap(colors[current], colors[end]);
            end--;
        } else {
            current++;
        }

    }
}
int main() {
    cin >> n;
    for(int i = 0; i < n; ++i) {
        cin >> colors[i];
    }
    partition(n);
    for(int i = 0; i < n; ++i) {
        cout << colors[i];
    }
    return 0;
}
//LeetCode
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int begin = 0, cur = 0, end = nums.size()-1;
        while(cur <= end) {
            if(nums[cur] == 0) {
                swap(nums[cur], nums[begin]);
                begin++;
                cur++;
            } else if(nums[cur] == 1) {
                cur++;
            } else {
                swap(nums[cur], nums[end]);
                end--;
            }
        }
    }
};
